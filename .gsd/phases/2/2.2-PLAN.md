# Plan 2.2: Ticker Input and Prediction UI

## Objective
Build the core interaction components where the user inputs a stock ticker, selects a timeframe, and views the model's directional prediction alongside confidence metrics.

## Context
- .gsd/SPEC.md
- .gsd/REQUIREMENTS.md (REQ-01, REQ-06, REQ-07)
- backend/app/routers/prediction.py (for API payload shapes)
