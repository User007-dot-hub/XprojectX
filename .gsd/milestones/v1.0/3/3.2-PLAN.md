# Plan 3.2: Frontend Chat UI

## Objective
Build a premium, slide-over chat interface on the frontend integrating with the new `/api/chat` endpoint.

## Context
- .gsd/SPEC.md
- frontend/src/App.jsx

## Tasks

<task type="auto">
  <name>Create ChatPanel Component</name>
  <files>frontend/src/components/ChatPanel.jsx, frontend/src/components/ChatPanel.css</files>
  <action>
    - Build a slide-out side panel (from the right) for the chat interface.
    - Use `lucide-react` (e.g. MessageCircle, X, Send) for icons in the header and input area.
    - Maintain local state for message history: `[{role: 'user' | 'assistant', content: str}]`.
    - Add a styled input area with a Send button.
    - Style messages distinctly (user messages on right, assistant on left). Ensure a dark, glassmorphism aesthetic strictly matching the App.css variables.
    - When user sends, append to local state, show loading spinner, `POST` to `http://localhost:8000/api/chat`. On success, append response. Handle API errors.
  </action>
  <verify>Verify component mounts and makes API requests correctly.</verify>
  <done>Chat panel renders properly, history maintains sequence, messages are formatted cleanly.</done>
</task>
